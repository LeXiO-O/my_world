## Preview

![image alt](https://github.com/LeXiO-O/my_world/blob/7d46dadbc29a59311eb0512eafbbc638f7e880fc/my_world_preview.png)
⚠️ The commands shown in this image are not updated ⚠️:
- Ctrl + Up / Down: rotate the view vertically.
- Ctrl + Left / Right: rotate the view horizontally.
- Ctrl + Z / S: zoom in and zoom out.

You can load the maps we pushed to the `map_files` directory.

The `vertex up` and `vertex down` buttons allow you to modify the shape of the map.
